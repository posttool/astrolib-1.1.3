Kugel's AstroLib main package.

This package contains the main class of Kugel's AstroLib, the TextHoroscop. This package is the center of all other packages. All possible house systems and planet calculation classes are used here.

Kugel's AstroLib is based on Michael Erlewine's "Manual of Computer Programming for Astrologers" (Matrix Software) and some source code of Walter 'Cruiser1' Pullen's "Astrolog 5.40" (http://www.astrolog.org). Some sources feature bibliographic references like (1315 [1], 51), referencing pages from Michael Erlewine's book or Walter 'Cruiser1' Pullen's code.
