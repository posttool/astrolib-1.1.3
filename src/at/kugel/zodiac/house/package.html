Kugel's AstroLib house system calculation algorithms.

This package contains the different house systems of Kugel's AstroLib. This package is only dependant on the different constants and the util package. All house system calculation algorithms are subclasses of HouseBasic which itself implements HouseInt. New house systems must implement HouseInt and may wish to extend HouseBasic to inherit common functionality, but are not forced to do so.
